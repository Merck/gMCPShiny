# Card with header

Card with header (Bootstrap 5).

## Usage

``` r
headerCard(title, ...)
```

## Arguments

- title:

  Card title.

- ...:

  List of elements to include in the body of the card.

## Value

Card element with header and body.

## Examples

``` r
if (interactive()) {
  headerCard("Card title", "Card body")
}
```
