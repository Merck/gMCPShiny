# gMCPShiny: Shiny App for Graphical Multiplicity

A 'Shiny' app for 'gMCPLite' for graphical described multiple test
procedures introduced in Bretz et al. (2009)
[doi:10.1002/sim.3495](https://doi.org/10.1002/sim.3495) and Bretz et
al. (2011)
[doi:10.1002/bimj.201000239](https://doi.org/10.1002/bimj.201000239) .
Contains instructions of multiplicity graph and graphical testing for
group sequential design, described in Maurer and Bretz (2013)
[doi:10.1080/19466315.2013.807748](https://doi.org/10.1080/19466315.2013.807748)
. Implements multiplicity graph visualization and reproducible code
generation.
